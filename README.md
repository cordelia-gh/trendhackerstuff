# TrendHacker Skibidi Shield

Changes made:

- Audio code now looks for `skibidi-toilet-song.mp3`.

To use the Skibidi Toilet song:
1. Get a legal MP3 copy of the audio.
2. Rename it exactly to `skibidi-toilet-song.mp3`.
3. Put it inside this extension folder, beside `manifest.json`.
4. Go to `chrome://extensions` and click Reload on the extension.
